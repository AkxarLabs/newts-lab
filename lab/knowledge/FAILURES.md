# Failures

What was **tried** and didn't work, with the diagnosed reason — experiments, approaches, investigations. Ideas killed before any experiment (non-novel, out of scope) belong in their IDEA.md + OPEN-QUESTIONS, not here. This file is what makes the next project smarter — record failures with the same care as findings. Append-only; newest first.

Entry format:

```markdown
## X-NNN: <one-line description of what failed>  (YYYY-MM-DD, project: <slug>)
**What was tried:** the approach/experiment, with config/run pointers.
**What happened:** the observed outcome (numbers, errors), with artifact pointers.
**Diagnosis:** best current explanation — implementation bug? wrong hypothesis? insufficient scale? confound?
**Transferable lesson:** what a future project should do differently (or avoid retrying).
```
