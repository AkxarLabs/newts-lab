# Papers

One directory per paper: `papers/<slug>/`, created by `/write-paper` from `templates/paper/`.

```
papers/<slug>/
├── main.tex          # the paper
├── references.bib    # citations — only from the idea's lit-review notes
├── figures/          # generated by scripts in the project repo, copied here
├── claims.yaml       # every quantitative claim → run id + artifact path (audited by /review-paper)
└── reviews/          # internal review reports
```

Build with `latexmk -pdf main.tex` (build artifacts are gitignored).
